Project Tracker

An AI-assisted project management dashboard designed to work with Claude Code.

How it works

  • The dashboard is a self-contained HTML file with embedded JSON data
  • A companion Claude Code skill (SKILL.md) manages the data through conversation
  • Three layers: memory files (source of truth) + JSON (structured data) + this dashboard (rendered view)

Claude Code Skill

The corresponding skill file (SKILL.md) teaches Claude Code how to manage this dashboard. It handles:

  • Adding/updating projects, notes, action items, and tickets
  • Syncing data from Jira and other tools
  • Drafting outreach messages and storing them as artifacts
  • Tracking current work focus with the "Currently Working On" feature
  • Regenerating the dashboard after any data change

Data Management

Use the Settings button to export, import, or directly edit the underlying JSON data. The JSON is embedded in the HTML file itself - no server required.
